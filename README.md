# :construction_worker: Como Rodar?
```bash
# Clone o Repositório:
$ git clone https://github.com/KaikySantos/youtube-clone.git
# Entre na pasta criada:
$ cd youtube-clone
# Instale as dependências:
$ npm install
# Rode a aplicação:
$ npm start
```
Acesse http://localhost:3000/ para ver a aplicação!
